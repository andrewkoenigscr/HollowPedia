⚔️ HollowPedia - Uma Enciclopédia de Hallownest ⚔️

HollowPedia é um fã-site e wiki dedicado ao aclamado jogo Hollow Knight, desenvolvido como projeto acadêmico para a disciplina de Desenvolvimento Web Básico da Universidade Positivo.

O site foi construído do zero, utilizando apenas HTML5, CSS3 e JavaScript (Vanilla JS), com foco em boas práticas, semântica e componentização para criar uma experiência de usuário imersiva.

✨ Principais Características

O projeto demonstra os seguintes conceitos:

HTML Semântico: Estrutura organizada e acessível, utilizando corretamente tags como <header>, <footer>, <main>, <section>, <aside>, <table> (para os amuletos) e <form> (para o cadastro).

CSS Moderno: Estilização avançada com:

Flexbox para o layout das caixas de personagens e regiões.

Animações CSS (@keyframes) para o efeito de partículas (cinzas) caindo em tela.

Efeitos de Hover e transições suaves para interatividade.

JavaScript (Vanilla JS):

Componentização Dinâmica: O rodapé (footer.html) e o modal de suporte (modal.html) são arquivos separados, carregados dinamicamente em todas as páginas usando fetch() e DOMParser, evitando repetição de código (princípio DRY).

Interatividade: Um modal de suporte funcional e reutilizável em todo o site.

Conteúdo Completo: Seções detalhadas que cobrem:

A Lore principal do jogo.

Listagem de todos os NPCs e Chefes.

Descrição de todas as Regiões.

Uma tabela completa com todos os 40 Amuletos.

Formulário Acessível: Uma página de cadastro com um formulário semântico completo, incluindo <label>, <fieldset>, <legend> e 7+ tipos diferentes de input.
